  - Delete (Delete button on products list)

  - Read (View at /admin/articles)

  - Delete (Delete button on articles list)

All data saved to Neon PostgreSQL
User-scoped (only see your own products/articles)

TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────
Admin redirects to login?
  → You're not logged in. Go to /sign-in first

Can't create account?
  → Go to /sign-up first, then /sign-in

Products not saving?
  → Check if logged in. All products require authentication.

Animations not visible?
  → Try scrolling on homepage. Animations trigger on scroll.

Can't see scroll animations?
  → Try on mobile view (375px width) for better effect visibility
